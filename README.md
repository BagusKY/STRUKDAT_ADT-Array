# TUGAS STRUKTUR DATA

## ADT Array
Implementasi Array satu dimensi dengan ukuran tetap.

Operasi:
- length()
- getitem
- setitem
- clearing()
- iterator()

## Game of Life
Simulasi Game of Life menggunakan ADT Array sebagai struktur grid 2D.

Aturan:
1. Sel hidup dengan 2 atau 3 tetangga tetap hidup
2. Sel hidup dengan <2 atau >3 mati
3. Sel mati dengan 3 tetangga menjadi hidup
